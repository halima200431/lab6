# 🍕 PizzaRecipes

## 📱 Description

Application Android en Java affichant une liste de pizzas avec image, prix et durée, avec accès à un écran de détails complet.

## 🚀 Fonctionnalités

* Splash Screen
* Liste des pizzas (ListView + Adapter)
* Détail (ingrédients, description, étapes)

## 🧠 Technologies

* Java
* Android Studio
* XML
* ListView + Adapter
* DAO + Service

git clone https://github.com/ton-username/PizzaRecipes.git
